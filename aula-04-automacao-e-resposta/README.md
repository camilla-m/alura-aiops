## 🧩 Habilidades desenvolvidas

- Planejar ações reativas sistêmicas acionadas por gatilhos de alertas de observabilidade
- Codificar rotinas automáticas de reinicialização de serviços, rollbacks e escalonamentos de infraestrutura
- Configurar o envio automático de resumos de incidentes e sugestões de ações para as salas de crise
- Implementar travas de segurança e cenários de human-in-the-loop para conter o raio de impacto de automações
- Criar uma automação ponta a ponta que execute uma ação com base no diagnóstico recebido
