# Vídeo 5.5 — Hands-on: Fluxo completo de AIOps

## 🎬 Roteiro

**Tipo:** Prática Guiada — Capstone  
**Habilidade:** Orquestrar todos os conceitos do curso em um fluxo de incidente completo, do alerta à documentação.

### Desenvolvimento

1. **Exercício 1:** Simular um incidente de produção com múltiplos sinais
2. **Exercício 2:** Passar pelo pipeline completo (correlação → diagnóstico → guardrails → remediação)
3. **Exercício 3:** Gerar a comunicação automática para stakeholders
4. **Exercício 4:** Produzir o post-mortem e o runbook atualizado
5. **Reflexão final:** Como aplicar AIOps no seu contexto real

## 💡 O que você vai praticar

- Orquestrar todos os módulos do curso em sequência
- Ver o fluxo completo de AIOps em ação: sinal → resolução → documentação
- Identificar onde aplicar cada técnica em ambientes reais
- Gerar artefatos prontos para uso: post-mortem, runbook e comunicações
